switchEthereumChain

  • switchEthereumChain(chainId: string, chainParams?: AucEthChainParams): Observable<boolean>
    • Defined in lib/services/wallet-connect/wallet-connect.service.ts:334

    This method is used for switching Metamask network by {@link chainId}.
    If Metamask doesn't have this {@link chainId} it can be added by parameter chainParams.

    Parameters

    • chainId: string

      0x-prefixed hexadecimal string.
      If you don't have chainId, you can use helper aucConvertChainIdToHex to generate it.

    • Optional chainParams: AucEthChainParams

      An optional parameter.
      Uses for adding new network to Metamask if it doesn't include network by {@link chainId}.
      This Library already have chainParams for the next chainIds AUC_CHAIN_ID.
      You can use your own specific params.

    Returns Observable<boolean>
